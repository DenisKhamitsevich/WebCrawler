# WebCrawler
WebCrawler - это веб-приложение, представляющее собой поискового робота, который начинает поиск с заданного URL и переходит по всем найденным ссылкам, чтобы обнаружить заданные термины и собрать статистику о их количестве.

![3](https://user-images.githubusercontent.com/61109220/159238897-09557bdd-d6fd-4641-beab-17300b2f25a7.png)

Для работы программы пользователь должен ввести:
- стартовый URL
- список терминов(разделяя их символом ";")
- максимальное количество просматриваемых страниц
- глубину поиска
